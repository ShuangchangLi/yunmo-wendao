# 云墨问道

一个武侠修仙题材的网页卡牌 roguelike MVP。以牌为招，以真气为势，在水墨山河里入山、破劫、悟道。

## Play

After GitHub Pages is enabled, the live build will be available from the repository Pages URL.

## Run

```bash
npm start
```

Then open `http://127.0.0.1:5173`.

## MVP Features

- Wuxia cultivation theme
- Turn-based card combat
- Character select
- Spirit energy, Qi, Transcendence, guard, and burn
- Enemy intents and cultivation-themed encounters
- Card rewards between fights
- Restartable run loop

## Tech

- HTML
- CSS
- Vanilla JavaScript
- Static site friendly, suitable for GitHub Pages
